///////////////////////////////////////////// PS KILL /////////////////////////////////////////////

La commande "ps" est pour afficher les processus en cours d'exécution.

- ps

Pour voir tous les processus en cours de mes users :

- ps -ef

Pour voir les processus parents ainsi que leurs enfants :

- ps -fax

La commande "kill" est pour arrêter un processus :

- kill 5454

** 5454 est le PID de mon processus pour indiquer lequel je veux arrêter **
